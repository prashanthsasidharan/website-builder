
## Drag & Drop Website Builder

A user-friendly tool for designing websites with drag-and-drop elements. Customize element attributes, preview designs, save your work, and easily rearrange components for a seamless web creation experience.

demo link - https://website-builder-prash.netlify.app/

## Tech Stack

**Framework** - react

**Drag and drop** - react-beautiful-dnd

**Styling**- Chakra UI




## Features

- **Intuitive Drag & Drop:**  
  Easily add elements to your canvas by dragging and dropping.

- **Customizable Elements:**  
  Configure individual attributes such as image URLs, alt text, and more.

- **Real-Time Preview:**  
  Instantly view your design as you build it.

- **Save & Resume:**  
  Save your layout data and reload it later for further editing.

- **Dynamic Reordering:**  
  Seamlessly move and arrange elements to perfect your design.

## Run Locally

Clone the project

```bash
  git clone https://github.com/prashanthsasidharan/website-builder.git
```

Go to the project directory

```bash
  cd website-builder
```

Install dependencies

```bash
  npm install
```

Start server

```bash
  npm start
```

View the app using http://localhost:3000/ in browse tab
